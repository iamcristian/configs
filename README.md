# ConfigsLinux



## Install vim-plug
https://github.com/junegunn/vim-plug

```sh
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Add root
```sh 
ln -s /home/user/.vim ./.vim
```
```sh 
ln -s /home/user/.vimrc ./.vimrc
```
